Sobre

Monitoramento de Silos

O projeto tem como objetivo o monitoramento das condições internas de silos de cereais.

Dados Cliente

Usuário
Identificação do cliente, dado único e não alterável.

Senha
Senha do cliente, dado necessário para o controle de acesso, alterável.

 Dados Silo

Nome do Silo
Identificação do silo, dado alterável.

Código de Série
Texto que identifica o Arduíno localizado no silo, dado único e não alterável.

Tipo de material
Tipo do grão armazenado no silo necessário para realizar as classificações, dado alterável.

 Descrição do Relatório

Relatório
Opção de relatório mensal e diário sobre o sistema. Está função está disponível apenas para clientes devidamente logados.

Opção
Seleciona a característica que se deseja visualizar no gráfico.
